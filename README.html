Basic Project

Name: Basic-Project Version: 1.0.0
Author: Ken Torry

A simple starter template, for modern Front-End Development. Uses require.js to load a starter project for working in Angular.js , Backbone.js or plain ole Jquery. It also uses SCSS(Sass) for css complilation.

** Inside the index.html document, change the data-main path to load the resource in the script tag **

  • To load Jquery only use : config-jquery
  • To load Angular use : config-angular
  • To load Backbone use : config-backbone
